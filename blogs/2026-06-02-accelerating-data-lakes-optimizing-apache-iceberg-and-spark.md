Google Cloud announced gcs-analytics-core, a new open-source Java library designed to centralize and accelerate analytics optimizations for Google Cloud Storage. Available natively in Apache Iceberg Java runtime starting from version 1.11.0, the library introduces Vectored I/O for parallel data fetching and Smart Parquet prefetching to reduce metadata retrieval overhead, with plans to expand to additional analytics engines by end of year.
