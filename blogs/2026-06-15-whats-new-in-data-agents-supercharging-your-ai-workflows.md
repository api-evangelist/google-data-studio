Google Cloud introduces new data agents and tools across its Agentic Data Cloud, spanning Conversational Analytics over BigQuery, AlloyDB, Spanner, Cloud SQL and Looker, plus agents including the Data Engineering Agent (GA), Data Science Agent, Database Observability Agent, and Looker Dashboard Agent (preview). Developer tooling includes the Data Agent Kit (preview), Managed MCP Servers for Databases (GA), MCP Toolbox for Databases 1.0 (GA), and QueryData for Cloud SQL, AlloyDB, and Spanner.
