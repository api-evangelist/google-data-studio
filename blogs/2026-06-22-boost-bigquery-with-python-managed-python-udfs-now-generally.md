Google Cloud announced the general availability of BigQuery Managed Python User-Defined Functions (UDFs), letting data practitioners run custom Python code directly inside BigQuery SQL with automatic scaling to billions of rows. The release adds vectorized PyArrow processing for up to 10x faster performance, configurable container resources up to 16 GB memory and 4 vCPUs, and billing through the BigQuery Services SKU with commitment-based discounts.
